## 📌 Project Description

The INES Event Ticket Generator is a full-stack web application developed to solve real problems faced by event organizers and students:

- Slow registration process
- No unique ticket reference
- Manual attendance lists
- Difficult check-in verification

The system allows:

- Attendee registration
- Unique ticket code generation (server-side)
- Ticket storage in MySQL
- Ticket lookup
- Admin dashboard statistics
- Edit/update functionality
- Check-in tracking
